# Phonebook App

Contains two pages:
- List
- Create

## List

To display all contacts created with their phone numbers

## Create

To create contacts. Contains Name and Phone Number fields

## Setup

1. Run `npm run install:all`
2. Run `yarn serve` or `npm start`

Note: The backend API is deployed at Heroku(https://phonebook-axrail-api.herokuapp.com/). The backend codes can still be found here.
